## 📥 Como Clonar e Executar

### Pré-requisitos

- Um navegador web moderno (Chrome, Firefox, Safari, etc.)
- Git instalado no seu sistema

### Passos para Clonar

1. Abra o terminal ou prompt de comando.
2. Execute o comando abaixo para clonar o repositório:

   ```bash
   git clone https://github.com/Wilian96/Tela-de-Login-DevClub.git
   ```

3. Navegue até a pasta do projeto:

   ```bash
   cd Tela-de-Login-DevClub
   ```

4. Abra o arquivo `index.html` no seu navegador preferido.

   - Você pode simplesmente clicar duas vezes no arquivo `index.html` ou
   - Usar um servidor local para uma melhor experiência (opcional).
